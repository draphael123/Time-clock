### Tools You Can Use:
1. **Screenshots:**
   - Take screenshots of your extension
   - Use Chrome's screenshot tool
   - Or use Snipping Tool (Windows) / Screenshot (Mac)

2. **Image Editing:**
   - **Free:** Canva, GIMP, Paint.NET
   - **Paid:** Photoshop, Figma
   - **Online:** Photopea (free Photoshop alternative)

3. **Promo Tiles:**
   - Use Canva templates
   - Create in Figma
   - Design in Photoshop

### Screenshot Tips:
- Show the extension in actual use
- Add text annotations if helpful
- Use consistent styling
- Highlight key features
- Make it visually appealing

### Promo Tile Tips:
- Use your brand colors
- Keep text readable
- Show key features
- Make it eye-catching
- Match your extension's style
