# Chrome Web Store Assets

This folder contains templates for creating all required assets for Chrome Web Store submission.

## üì∏ How to Create Screenshots

### Method 1: Using HTML Templates (Recommended)

1. **Open the HTML files in Chrome:**
   - `screenshot-template.html` - Main extension view
   - `screenshot-dark-mode.html` - Dark mode view

2. **Take Screenshots:**
   - Press `F12` to open DevTools
   - Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
   - Type "Capture screenshot"
   - Select "Capture full size screenshot"
   - Save as PNG

3. **Resize if needed:**
   - Required size: 1280x800 or 640x400 pixels
   - Use image editor to resize if necessary

### Method 2: Screenshot Your Actual Extension

1. **Open your extension popup**
2. **Take screenshot:**
   - Windows: `Win+Shift+S` (Snipping Tool)
   - Mac: `Cmd+Shift+4`
   - Chrome: Right-click ‚Üí Inspect ‚Üí Device Toolbar ‚Üí Screenshot

3. **Edit screenshot:**
   - Add text annotations
   - Add feature highlights
   - Use image editor (Canva, Photoshop, GIMP)

### Small Promo Tile (440x280) - REQUIRED

1. **Open `promo-tile-small.html` in Chrome**

1. **Open `promo-tile-marquee.html` in Chrome**

   - Use browser screenshot tool
   - Or use DevTools screenshot feature
3. **Save as:** `promo-tile-marquee.png`
