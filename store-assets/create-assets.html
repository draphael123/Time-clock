How to Create Assets

  1. Click the links below to open each template
  2. Take a screenshot using Chrome DevTools:
    • Press F12 to open DevTools
    • Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
    • Type "Capture screenshot" and select "Capture full size screenshot"
  3. Save screenshots as PNG files
  4. Use these files when submitting to Chrome Web Store
